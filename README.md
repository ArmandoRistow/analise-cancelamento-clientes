# 📊 Análise de Cancelamento de Clientes

## 📌 Descrição
Projeto de análise de dados para identificar os principais motivos de cancelamento de clientes.

## 🚀 Tecnologias
- Python
- Pandas
- Matplotlib

## 📈 Objetivo
Identificar padrões de cancelamento e propor melhorias para reduzir a taxa de churn.

## 🎯 Resultados
Possibilidade de redução da taxa de cancelamento de 56% para 18%.
